| [Position](#position){#position}                                                                | Job title of the employee                                                     | Mandatory                                                                                                                                                                                                                        | Job                              |

| [Team](#team){#team}                                                                            | Team or department the employee belongs to                                    |                                                                                                                                                                                                                                  |                                  |

| [Rate](#rate){#rate}                                                                            | Pay rate for the employee                                                     |                                                                                                                                                                                                                                  | Job                              |

Default: 0                                                                                                                                                                                                                 |                                  |

Default: 0                                                                                                                                                                                                                  |                                  |
