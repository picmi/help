# KeyLink integration overview

The KeyLink integration allows PICMI to generate CSV files formatted for ingestion by the Platinum Payroll system.

## Integration configuration

| Configuration                                                                 | Description                                                                  | Values                                                                                  |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [Payroll Group](#cfg-payroll-code){#cfg-payroll-code}                         | The payroll identifier used by KeyLink to determine which pay rules to apply | Text (required)                                                                         |
| [Employment Type](#cfg-employment-type-default){#cfg-employment-type-default} | Default “Employment Type” when not specified                                 | KeyLink codes such as FIX, CAS, PRU                                                     |
| [Province](#cfg-province){#cfg-province}                                      | When the public holiday is for the region                                    | Regions                                                                                 |
| [Leave Set](#cfg-leave-set){#cfg-leave-set}                                   | Default roster configuration                                                 | Text                                                                                    |
| [Employer Subsidy Contribution Tax Rate](#cfg-subsidy-tax){#cfg-subsidy-tax}  | ESCT tax rate on employer subsidy contribution                               | 10.5%, 17.5%, 30%, 33%, 39%                                                             |
| [Employer Subsidy Rate](#cfg-subsidy-rate){#cfg-subsidy-rate}                 | Rate of employer subsidy contribution                                        | Percentage number (e.g., 3)                                                             |
| [Convert name to simple characters](#cfg-convert-name){#cfg-convert-name}     | Used to ensure bank account names are acceptable to banks on integration     | Leave name as typed by the job seeker; Send name without macrons and special characters |
