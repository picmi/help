# Search contract templates

PICMI allows you to find contract templates quickly by using search, workflow type filters, and table columns. Here are the [general search filter instructions](search.md).

:::: explanation
## Search fields

* **Search Templates**: searches by the contract template name
* **Workflow Type**: filters templates by workflow use, such as employment or service
* **More Filters**: opens additional filter options
* **Clear All**: removes all current filters

## Columns

The contract template list can show different columns, such as description, archived status, last modified, and workflows.

::: prompt
If a template does not appear, clear the filters first and then search again.
:::
::::

:::: instructions
## Search for a contract template

1. Locate **Menu**, select **Contracts**
2. Click in **Search Templates**
3. Type part of the template name
4. Review the filtered list

The list will update to show only matching templates.
::::

:::: instructions
## Filter by workflow type

1. Click **Workflow Type**
2. Select **Employment Only** or **Service Only**
3. Click **Apply**

The list will refresh to show only templates matching that workflow type.
::::

:::: instructions
## Use more filters

1. Click **More Filters**
2. Choose the filter values needed
3. Click **Apply**

The list will refresh using the selected filters.
::::

:::: instructions
## Clear the filters

1. Click **Clear All**

All current search and filter selections will be removed.
::::

:::: instructions
## Customise the columns

1. Click the settings button on the contract template list
2. Select or clear the columns you want to show
3. Click **Save**

The table will update to show the selected columns.

::: prompt
Use the **Workflows** column when checking where a template is already in use before making changes.
:::
::::
