# Preview and edit a contract template

Contract templates can be opened in **Preview/edit** so you can review the template name, read the contract text, work with the inserted field chips, and move between the current version and older saved versions.

:::: explanation
## What you can review

* **Contract Template**: the template name is shown at the top of the page
* **Contract text**: the full agreement text and any inserted chips
* **Current version**: the editable working version of the contract template
* **Version history**: older saved versions that can be previewed
* **Insert mode**: used to add field substitutions from the available field groups on the editable current version
* **Navigate Inserted**: used to click through the chips already placed in the contract and jump to their locations in the text

::: prompt
Changes to a contract template affect future contracts created from that template, not contracts that have already been completed.
:::
::::

:::: instructions
## Open a contract template

1. Locate **Menu**, select **Contracts**
2. Locate the contract template row :::icon checkbox-marked-outline:::
3. Click :::icon dots-vertical::: to open the menu
4. Select **Preview/edit**

The contract template editor will open.
::::

:::: instructions
## Review the contract template

1. Check the template name shown in the header
2. Read through the contract text
3. Check whether you are viewing the **Current** version or an older saved version
4. Use **Insert Fields** to add field substitutions when editing the current version
5. Use **Navigate Inserted** to review the fields already used

This view is used for both reviewing and updating the contract template.
::::

:::: instructions
## View the current version

1. Open the contract template using **Preview/edit**
2. In the header, locate the version history button :::icon history:::
3. Check for the row marked **Current**
4. Select **Current** if needed

The current version is the editable version of the contract template.

::: prompt
When viewing **Current**, the formatting toolbar and field insertion controls are available.
:::
::::

:::: instructions
## Edit the current version

1. Open **Current**
2. Click in the contract text to type or update content
3. Use the formatting toolbar as needed
4. Use **Insert Fields** to add or replace field substitutions
5. Click **Save** when finished

Changes can only be made on the current version.
::::

:::: instructions
## Open version history

1. Open the contract template using **Preview/edit**
2. Click the version history button :::icon history:::
3. Review the list of saved versions by date
4. Select the version you want to view

The selected version will open in preview mode.

::: prompt
Older versions are read-only. They are for reviewing what was saved at that time.
:::
::::

:::: instructions
## Review an older version

1. Open a version from **Version history**
2. Read the contract text and field substitutions
3. Compare it with the current version if needed

Older versions are shown in preview mode and cannot be edited.
::::

:::: instructions
## Understand preview mode on older versions

1. Open a saved version from the version history list
2. Review the contract text only
3. Note that field insertion controls are not available
4. Return to **Current** if changes are required

Preview mode is useful when checking what the contract template looked like at a previous point in time.
::::

:::: instructions
## Return from version history to the editable version

1. Click the version history button :::icon history:::
2. Select **Current**

This returns you to the editable version of the contract template.
::::

:::: instructions
## Return to the contract template list

1. Click the :::icon arrow-left::: back arrow in the top left

You will return to the contract template list.
::::
