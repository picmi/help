# Duplicate an opportunity

An opportunity (employment job or service proposal) can be duplicated to make a copy. It will contain all the details of the previous one. The default change is to change the name newly created item. The application workflow remains the same. All fields that can be changed are specific to the new opportunity.

Note: intake capacity settings are not copied and must be set individually.

:::: instructions
## To create a duplicate

1. Go to either:
    * **Employment** > **Jobs**
    * **Services** > **Proposals**
2. Locate the row :::icon checkbox-marked-outline::: to duplicate
3. Click :::icon dots-vertical::: (vertical dots) to open menu
4. Select **Duplicate**
5. Click **Duplicate**

The new opportunity should be in the list (default "_Copy of ..._"), if you can't see it, click **Refresh**
::::

:::: instructions
## Making final changes

1. Go to either:
    * **Employment** > **Jobs**
    * **Services** > **Proposals**
2. On the newly duplicated item, locate the row :::icon checkbox-marked-outline:::
3. Click :::icon dots-vertical::: (vertical dots) to open menu
4. Select **Edit**
5. Move through the [fields](opportunity-create.md#fields), update values and **Save** as needed
::::
