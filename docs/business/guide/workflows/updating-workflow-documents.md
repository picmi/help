# Updating Workflow Documents (files and videos)

As your business evolves, you'll need to update documents and videos within your workflows. This guide walks you through
common updates you'll make to keep your workflows current and compliant.

If you are **UPDATING** a document or video, it is likely that you need ONLY update the **original document with a new
version** as the link will not change.

::: prompt
Ensure that you:

* Understand [workflows](../../about-picmi/understanding-workflows.md)
* Understand [items on workflows](../../about-picmi/understanding-workflow-items.md)

Need to create a new workflow? Check out [creating a workflow](creating-a-workflow) first.
:::

## Replacing a document in your workflow

When you need to update an existing document (like an Accommodation Agreement or Staff Handbook) with a new version,
follow these steps.

### Step 1: Prepare your new document

Before updating your workflow, get your document ready to use.

:::instructions

1. Ensure your new document is saved and approved
2. Keep the file name clear and descriptive
   :::

### Step 2: Locate the workflow

Find each workflow(s) that contains the document you need to replace.

:::instructions

1. Go to **Setup** â†’ **Workflows**
2. Open each workflow that might contain the document
3. Look through sections like Health & Safety, Induction, or Terms & Conditions
4. Note which section contains the old document
   :::

::: prompt
Not sure which workflows contain your document? Open each active workflow and use the preview feature to check what
applicants see.
:::

### Step 3: Update the document link or content

There are two options: update the link (in PICMI) or upload a new version (of the file or video in other system)

:::instructions
**Option 1** 

Replace the link of the old document with your new one.

1. Open the workflow in edit mode
2. Navigate to the section containing the document
3. Locate the document link or embedded file
4. Replace with your new document link or upload
5. Update any accompanying text if needed
6. Click **Save**

**Option 2**

Replace the content of the old document with your new version.

1. There are no changes to the workflow itself.
2. In the other system, update either:With update the file or video in other system.
3. Upload the new version to Google Drive
    * [File](../../workflows/linked-files.md#updating-files)
    * [Video](../../workflows/linked-videos.md#updating-files)
:::

See [updating workflows](../../workflows/updating-workflows) for detailed editing instructions.

### Step 4: Preview and verify

Always check your changes before applicants see them.

:::instructions

1. Click **Preview** in the workflow editor
2. Navigate to the section with your updated document
3. Click the document link to ensure it opens correctly
4. Verify the correct version appears
   :::

::: prompt
Testing takes 2 minutes and prevents applicants from seeing broken links or wrong documents.
:::

## Frequently asked questions

:::faq Document uploaded but not showing?

Uploading a document to Google Drive doesn't automatically add it to your workflow. You must explicitly add the link or
embed it in the workflow editor.
:::

:::faq Link not working for applicants?

Check your document's sharing permissions. Set links to "Anyone with the link can view" if you want all applicants to
access them.
:::
