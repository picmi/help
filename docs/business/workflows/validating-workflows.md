# Validating workflow changes

After updating a workflow, always validate it's working correctly.

:::instructions

1. Go to **Employment** → **Jobs** (or **Services** → **Proposals**)
2. Locate a job using the updated workflow
3. Click **⋮** (vertical dots) on the job row
4. Select **Check Configuration**
5. Confirm validation passes :::icon check primary:::
6. Fix any errors :::icon exclamation::: before sending new invites
   :::
