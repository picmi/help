# Reviewing and exporting workflow contents

Use workflow **Review** mode to audit workflow contents in a table.

Review mode helps you check what is in a workflow without moving through it as a participant.

::: prompt
**Review mode** is different from a routing **Review outcome**. Review mode audits workflow contents. A routing Review
outcome means a participant answer needs human review before the workflow can continue.
:::

## Review workflow contents

::: instructions

1. Go to **Setup** > **Workflows**.
2. Locate the workflow row, click &vellip; (vertical dots) to open the menu.
3. Select **Review** mode.
4. Search the table to find specific pages, items, or text.
5. Group the table when you need to review related items together.
6. Check the workflow contents.

:::

Review mode can include:

- order
- page
- item type
- question type
- participant text
- description
- choices
- required state
- usage

## Copy or export review data

::: instructions

1. Go to **Setup** > **Workflows**.
2. Locate the workflow row, click &vellip; (vertical dots) to open the menu.
3. Select **Review** mode.
4. Filter, search, or group the table as needed.
5. Copy the review data, or download it as CSV.
6. Choose whether to remove HTML from exported descriptions.

:::

Remove HTML from exported descriptions when the CSV needs to be reviewed in a spreadsheet or shared with people who only
need the readable text.

Keep HTML when you need to inspect formatting or links stored in the description.
