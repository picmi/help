# Creating and editing Library Items from the Workflow Builder

Some Library Items can be created or edited while you are working in the Workflow Builder.

Library Items are reusable across workflows. Before editing one, check whether the same item is used elsewhere.

## Create a Library Item

::: instructions

1. Go to **Setup** > **Workflows**.
2. Locate the workflow row, click &vellip; (vertical dots) to open the menu.
3. Select **Edit** mode.
4. Open the **Library** panel.
5. Choose the type of Library Item to create.
6. Complete the required fields.
7. Add the new Library Item to the workflow.
8. Save the workflow.

:::

You can create supported items such as:

- Questions
- Information
- Content Blocks: Page Title, heading, video, image
- Contracts

## Edit a linked Library Item

::: instructions

1. Go to **Setup** > **Workflows**.
2. Locate the workflow row, click &vellip; (vertical dots) to open the menu.
3. Select **Edit** mode.
4. Select the page or item that links to the Library Item.
5. Review the linked item and usage details in the Inspector.
6. Edit the linked item if the builder allows editing for that type.
7. Save your changes.

:::

Use this for supported linked Questions and Content Blocks.

Some linked Library Item types may need to be edited from their main setup area. For example, some contract template
changes may need to be made under **Setup** > **Contracts**.

::: prompt
Editing a linked Library Item can affect every workflow using that item. If only one workflow needs different wording,
rules, content, or contract terms, create or link a separate Library Item instead.
:::
